# How to Use Ivy (5-Minute Guide)

## Goal
Turn thinking into reusable assets.

---

# Step 1 — Capture

Take a conversation or work session.

Create a snapshot:
- structured
- concise
- complete

Do NOT paste raw transcripts.

---

# Step 2 — Store

Save it:
- correct folder (snapshots/)
- correct filename
- valid metadata

---

# Step 3 — Extract

If something is reusable:
→ create an artifact

If something recurs:
→ create a concept

---

# Step 4 — Link

Connect:
- snapshots → concepts
- snapshots → artifacts

---

# Step 5 — Reuse

Before starting new work:
- search Ivy
- reuse prior thinking
- extend instead of restarting

---

# Golden rule

If you find yourself rethinking something,
it should already exist in Ivy.
